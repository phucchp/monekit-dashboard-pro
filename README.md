# Monekit Dashboard

Production-ready React dashboard starter built with:

- Vite
- React + TypeScript
- Tailwind CSS
- Redux Toolkit
- React Router
- ESLint + Prettier

## Getting started

```bash
npm install
npm run dev
```

## Available scripts

- `npm run dev` - start development server
- `npm run build` - type-check and build for production
- `npm run preview` - preview production build
- `npm run lint` - run ESLint
- `npm run format` - format files with Prettier
- `npm run format:check` - check formatting

## Deployment

This project is configured for automatic deployment to GitHub Pages.

1.  **Push to `main`**: Any push to the `main` branch will trigger the GitHub Actions workflow.
2.  **Build & Deploy**: The workflow will automatically build the project and deploy the contents of the `dist` folder to the `gh-pages` branch.
3.  **View Live**: The deployed site is available at [https://phucchp.github.io/monekit-dashboard-pro/](https://phucchp.github.io/monekit-dashboard-pro/).
